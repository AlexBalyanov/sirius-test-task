# Chip List Test Task

Тестовое задание: переиспользуемый React-компонент для отображения списка чипсов в одну строку.

## Что реализовано

- Чипсы отображаются в одну строку внутри родительского контейнера.
- Элементы, которые не помещаются в строку, скрываются за кнопкой `...`.
- По клику на `...` открывается попап с оставшимися чипсами.
- Список автоматически пересчитывается при изменении ширины контейнера.
- Поддерживается множественный выбор чипсов.
- `Chip` можно использовать отдельно от списка.
- `ChipList` можно переиспользовать с любым массивом элементов формата `{ id, label }`.
- На демо-странице есть input для изменения ширины контейнера.

## Технологии

- React
- TypeScript
- Vite
- CSS Modules
- Radix UI Popover

`@radix-ui/react-popover` используется только для попапа.

## Особенности реализации

`ChipList` измеряет доступную ширину списка через `ResizeObserver`. Для корректного расчета используется скрытый измерительный список: он позволяет получить реальные ширины всех чипсов и кнопки `...`, включая те элементы, которые сейчас не отображаются в основной строке.

На основе этих размеров компонент вычисляет максимальное количество чипсов, которое помещается в одну строку. Остальные элементы рендерятся внутри попапа.
